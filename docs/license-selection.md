# License Selection

NodeStatus now uses the `MIT` license.

## Why MIT

- It keeps contribution friction low.
- It is easy for individual and company contributors to understand.
- It matches the current goal of growing a community around adapters, rule packs, fixtures, and Android client improvements.

## Tradeoff accepted

MIT does not include an explicit patent grant like Apache-2.0.
That is acceptable for the current phase of this project.

## Maintenance note

If ownership wording changes later, update the copyright line in the root [LICENSE](../LICENSE).
