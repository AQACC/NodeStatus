# Rule Packs

Rule packs are the intended community contribution surface for provider-specific behavior.

## What belongs here

- pack manifests
- capability declarations
- sanitized fixtures
- future extraction and mapping rules

## What does not belong here

- real credentials
- raw customer data
- executable scripts intended to run on Android devices

The sample pack under `packs/virtfusion.vmvm/` is a shape reference, not a finished adapter.
